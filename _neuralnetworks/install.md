動作環境
-----------------------------------------
Rindow NeuralNetworksは以下の動作環境でテストしています。

- PHP 7.2, 7.3, 7.4
- Windows 10
- Ubuntu 18.04, 20.04
- AMD CPU/APU 64bit(SSE2)

Intel CPUでも動作するでしょう。

PHPのインストール
-----------------------------------------
Windows10では、Windows版PHPをインストールします。

+ https://windows.php.net/download/ から PHP7.4 x64 Thread Safe版をダウンロードしてください。

# リリースノート: バージョン 1.1.0

## 新しい機能
- ネイティブなスレッドを使った並列処理ライブラリを追加しました。今後はOpenMP版ではなくスレッド版のライブラリの使用を推奨します。これは、WindowsのOpenMPライブラリがPHPのプロセス終了時に問題を引き起こすためと、並列処理アルゴリズムの効率化が理由です。

## 新しい関数
- `masking`: 数値の配列をブーリアン配列でマスクします。
- `cumsumb`: `cumsum`関数の拡張版です。
- `bandpart`（整数型対応）: `bandpart`関数に整数型が追加されました。
- `gatherb`: `gather`関数の拡張版です。
- `einsum`: `einsum`の汎用版です。処理可能な数式の自由度は高いですが低速です。
- `einsum4p1`: 機能限定版の`einsum`関数です。4+1次元の配列しか計算できませんが高速です。

## プレビュー版関数
将来インターフェースが変更されるか削除される可能性があります。
- `topk`: `top_k`関数です。
- `gathernd`: `gathernd`関数です。

## 下位互換性
- バージョン1.0と下位互換性があります。

## サポートプラットフォーム
- C++環境でソースコードからビルドできます。それぞれの環境でC++スレッド版とOpenMP版とシングルスレッド版があります。

### プレビルドバイナリー
- Ubuntu 24.04
- Ubuntu 22.04 (Ubuntu 24.04およびDebian 12で利用可能です)
- Windows （Windows 10/11上でVisual Studioのランタイムライブラリとともに使用できます。）
- macOS x86_64（Intel MacのmacOS 13で使用できます。）
- macOS Arm64（Apple SiliconのmacOS 14で使用できます。）

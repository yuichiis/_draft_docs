# リリースノート: バージョン 1.1.0

## 新しいPHPに対応
- PHP 8.4に対応しました。
- PHP 8.1,8.2,8.3,8.4に対応しています。

## 新しい関数
- `masking`: 数値の配列をブーリアン配列でマスクします。
- `cumsumb`: `cumsum`関数の拡張版です。
- `bandpart`（整数型対応）: `bandpart`関数に整数型が追加されました。
- `gatherb`: `gather`関数の拡張版です。
- `einsum`: `einsum`の汎用版です。処理可能な数式の自由度は高いですが低速です。
- `einsum4p1`: 機能限定版の`einsum`関数です。4+1次元の配列しか計算できませんが高速です。

## プレビュー版関数
将来インターフェースが変更されるか削除される可能性があります。
- `topk`: `top_k`関数です。
- `gathernd`: `gathernd`関数です。

## 下位互換性
- バージョン1.0と下位互換性があります。

## 対応プラットフォーム
- Windows 11、およびUbuntu 22.04/24.04でテスト済みです。
- Windows 10およびDebian 12でもおそらく動作します。
- macOSについてはこのモジュールは動作するようですが、rindow-math-buffer-ffiモジュールがmacOSで動作しない問題があるため十分他テストができていません。

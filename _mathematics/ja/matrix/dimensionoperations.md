概要
--------
MatrixOperator は、N 次元配列に対して一般的な演算を実行します。 ブロードキャストでは、すべての要素に同じ変更を与えることもできます。
乱数演算も提供します。
これらは Python の Numpy に似るように設計されています。

柔軟性よりも高速な動作を優先した機能も提供します。
これらは、Basic Linear Algebra Subprograms (BLAS) からの抜粋によって実装されています。 その他の科学計算も必要に応じて実装されます。

純粋なPHPだけでも動作します。OpenBLASやRindow-Matlibを呼び出して動作させることもできます。

概要
--------
PHP 用の数学ライブラリを提供するプロジェクト。

ほとんどの科学計算と機械学習ではベクトル演算が使用されます。
ただし、PHP 関数はスカラー演算にすぎません。
また、PHP Array はベクトル演算には適していません。

したがって、適切な Array オブジェクトを定義するのが基本方針です。
ベクトル操作用に、すべてをその上に構築します。

以下のような特徴があります。

- 共通の Array オブジェクト インターフェイス「NDArray」を定義します。
- 柔軟な行列演算ライブラリを提供します。
- 外部の演算ライブラリーをリンクして行列演算を高速化します。
- GPUを利用して行列演算を高速化します。
- 数学的データの視覚化ライブラリを提供します
